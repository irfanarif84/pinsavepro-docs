# Security Policy

Thank you for helping keep PinSavePro secure.

## Reporting a Security Issue

If you discover a security vulnerability, please report it privately.

Do not create a public GitHub issue for security-related problems.

## Contact

Website

https://pinsavepro.com

Support

https://pinsavepro.com/contact

## What to Include

Please include:

- Description of the issue
- Steps to reproduce
- Browser and operating system
- Screenshots (if applicable)
- Any supporting information

We will review all valid reports as quickly as possible.

Thank you for helping improve the security of PinSavePro.
